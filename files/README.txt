CV Download Setup Instructions
================================

To enable CV download functionality, follow these steps:

1. Place your CV file in this 'files' folder
2. Name it: "Imalsha_Hansinee_CV.pdf"
3. Update the file path in main.js if needed

Supported file formats:
- PDF (recommended)
- DOC/DOCX
- Any other format you prefer

The download button will:
✅ Show "Downloaded!" when successful
❌ Show "CV Not Found" if file is missing

Current expected file path: ./files/Imalsha_Hansinee_CV.pdf
